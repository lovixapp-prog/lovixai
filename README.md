# LOVIX AI

LOVIX AI is a React, Vite, TypeScript and Tailwind application for AI video, image, motion, UGC and influencer creation.

## Local Development

```sh
npm install
npm run dev
```

## Production

Production domain: https://lovix.app

Build locally with:

```sh
npm run build
```

Deploy to Cloudflare Pages with:

```sh
npx wrangler pages deploy dist --project-name lovix-app
```

## SEO Notes
